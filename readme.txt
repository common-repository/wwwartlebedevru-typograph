Русская типографика через сервис &laquo;Типограф&raquo; Студии Артемия Лебедева.


== Description ==

Исправляет русскую типографику: всякие "и" на &laquo;и&raquo;, и т.п. Подробнее о преобразованиях
можно прочитать на [странице Типографа](http://www.artlebedev.ru/tools/typograf/).

Плагин обрабатывает контент (`content`) и краткое описание (`excerpt`) всех постов и страниц. Результат
работы кешируется в файлах в папке `wp-content/art-typo-cache/`.

Т.к. для расстановки типографики требуется вызов внешнего веб-сервиса, время отклика сайта может
немного увеличиться, пока все результаты не будут закешированы.

== Installation ==

1. Распакуйте архив в папку `/wp-content/plugins/`. Убедитесь, что все php-файлы находятся в собственной
папке, а не в общем каталоге `/wp-content/plugins/`. 
1. Активируйте плагин через меню 'Plugins' в WordPress.
1. Всё! :)
